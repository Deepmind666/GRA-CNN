Abstract: Deep convolutional neural network is one of the most important deep learning models,
widely used in image recognition, speech recognition and other fields. In the research
and application of convolutional neural networks, the number of hidden layers and
nodes is generally determined by experience and expert knowledge. In this sense,
achieving optimal network architecture with excellent generalization capabilities and
low model complexity is time consuming and even impossible. In order to solve this
problem, this paper proposes a gray correlation analysis method to optimize the
convolutional neural network (GRA-CNN). The method evaluates the correlation
between hidden nodes and output nodes in the network model and selects hidden
nodes with higher correlation. Deleting hidden nodes with smaller degrees ensures that
the final convolutional neural network can have better recognition accuracy while
maintaining higher training performance and higher system stability and adaptability.
Experiments on several benchmark datasets show that the proposed method is
superior to many other methods in terms of model performance or model complexity.

 AB ST R ACT  
Deep convolutional neural network is one of the most important deep learning models, widely used in image 
recognition, speech recognition and other fields. In the research and application of convolutional neural networks, 
the number of hidden layers and nodes is generally determined by experience and expert knowledge. In this sense, 
achieving optimal network a rchitecture with excellent generalization capabilities and low model complexity is 
time consuming and even impossible. In order to solve this problem, this paper proposes a gray correlation 
analysis method to optimize the convolutional neural network (GRA -CNN). The method evaluates the correlation 
between hidden nodes and output nodes in the network model and selects hidden nodes with higher correlation. 
Deleting hidden nodes with smaller degrees ensures that the final convolutional neural network can have better 
recognition accuracy while maintaining higher training performance and higher system stability and adaptability. 
Experiments on several benchmark datasets show that the proposed method is superior to many other methods in 
terms of model performance or model complexity.

1. Introduction  
Deep learning is an active and exciting branch in machine 
learning and artificial intelligence research  [1], [2]. The intuition 
of deep learning originates from the biological research of human 
brains. Deep network simulates the connection and activation of 
nerve cells in a brain. By si gnificantly increasing the layers 
between input and output units, deep models obtain the ability of 
learning complexity functions, recognizing abstract concepts and 
processing many intelligent tasks. Deep learning models can 
learn essential characteristics  from massive data , sparking 
innovations across medicine and finance  [3], [4]. The successful 
applications of deep learning models are now changing people's 
lives and making our world increasingly intelligent  [5]-[7]. A 
systematic comparison between conventional feature -engineering 
pipelines and modern deep learning approaches further cla rifies 
this transition  [6]. 
On the other hand, since images are widely used in 
information transmission and knowledge sharing through 
internet, classification, reco gnition or grading become great 
challenge when building intelligent image processing 
applications. The low training and testing cost of the model are 
preferred especially in real -time and customer -friendly 
applications. The process of image recognition inc ludes image 
preprocessing, segmentation, feature extraction and model 
training  [8]. With the help of image recognition and 
classification, the fast retrieval of ima ge information can be 
achieved that has a widely range of application such as face 
recognition  [9], [10] safety protection  [11], autonomous driving 
[12], and medical image [13] and so on.  
The traditional image recognition model usually adopts a 
domain -specific feature extractor and a shallow model for 
recognition. Figure 1 sketches the main steps of a traditional 
image recognition model.   
 
Fig. 1. Main steps of traditional image recognition model  
Roughly speaking, traditional image recognition model 
generates clusters of similar images in the feature space induced 
by the feature representation, and achieves image recognition by 
classifiers such as KNN and SVM  [14]-[15]. For example, HOG 
features are widely used in pedestrian detection, LBP features in 
face recognition, and SIFT features in object recognition, etc.  
The mai n steps of image recognition based on deep learning 
are as follows. Raw pixels of a test image are directly fed to the 
deep network model. In the training stage, the network 
parameters, i.e. weights and biases, are updated according to the 
back -propagated training errors and the training rate. When the 
evaluation errors tend to be stable, the model converges and can 
be used as a recognition model  [16]. Figure 2 shows  the main 
steps of the deep learning -based image recognition model.  
 
 
  
Fig. 2. Main steps of the deep learning -based image recognition model  
Commonly, deep learning -based image recognition system 
includes feature extraction module and image recognition 
module. To achieve good performance, these two modules should 
be jointly trained and optimized. In this paper, we proposed an 
image recognition model based on convolutional neural network 
using gray relational analysis method (GRA -CNN).  
2. Related Works  
Machi ne learning models can be divided into shallow model 
and deep level. Traditional machine learning model usually 
adopts a shallow structure with 2 -3 layers. For example, 
Gaussian mixture model, K -means clustering, support vector 
machine are representative o f traditional machine learning [17]. 
These shallow models can achieve good results in solving simple 
practical problems, but they are not effective in expressing 
complex real -world problems [18]. 
From a narrow perspective, deep learning is a neural 
network that contains several hidden layers and has certain 
training methods and structures [19]. From a broad perspective, it 
is well accepted that any machine learning model with a deep 
hierarchical structure can be regarded as a deep lea rning model 
[20]. In the process of deep learning, the unsupervised learning 
method is used to process the input image and learn the features 
of the image. Then, th e supervising sequence is used to update 
the entire network structure parameters. Finally, the classification 
result of the image is output at the output layer. In recent years, 
with the rapid development of deep learning, it has been widely 
used in voice recognition, image classification, natural language 
processing and other fields.  
In 1974, Paul Werbos proposed a back -propagation (BP) 
algorithm to solve the linear inseparable problem of neural 
network applied to deep layers. However, when the number of 
network layers increase, BP neural network will also encounter 
problems such as "gradient disappearance" or instability of 
training structure, which leads to the inability of deep neural 
network [21]. A famous deep model called Deep Belief Network 
(DBN) was proposed and successfully applied to many 
classification tasks. A typical DBN model has a deep belief 
network by stacking several Restricted Boltzmann Machines 
(RBM), and solved the problems of gradient disappearance and 
local optimization by unsupervised layer -wise greedy algorithms 
[22]. Vincent et al. proposed the deep str ucture of the denoising 
Stacked Automatic Encoder (SAE) [23] to further improve the 
accuracy of unsupervised pre -training. LeCun proposed a 
Convolutional Neural Net work (CNN) based on DBN and SAE 
[24]. DBN, SAE and CNN constitute the three fundamental 
frameworks for deep learning and have made great achievements 
in natural ima ge processing and face recognition.

With the evolution of CNN architectures such as VGG net [20], 
deeper and more uniform convolutional layers demonstrated that 
increasing network depth can significantly improve image 
recognition performance, laying the foundation for later 
architecture optimization.  
3. The Proposed GRA -CNN Method  
3.1. Challenges of convolutional neural network construction  
The number of hidden nodes is  one of the most important 
factors for the network structure. The hidden feature map 
determines the number of hidden nodes in the network. The less 
hidden feature maps are the less information network obtains. 
The lower accuracy is the slower convergence r ate is.  On the 
contrary, the more hidden feature maps are, the higher accuracy 
of network is, but it maybe leads to training time too long and 
overtraining that result in worse recognition.  
In the process of research and application of convolutional 
neural  network, the hidden nodes are generally obtained by 
experience. By constructing and training a network structure that 
contains different numbers of hidden nodes, the best network 
structure is selected as the final network structure. This method 
of constru cting a network structure has limited capability for it is 
difficult to find an optimal network structure. In addition, in the 
convolutional neural network, the influence of different feature 
graph units on the output layer nodes is different, some of the 
feature graph units have a greater impact on the output nodes, 
and the other part of the feature graph units have a smaller one. 
Therefore, it is possible to improve the performance of 
convolutional neural network by deleting the unit with small 
influence without affecting the training effect.  Moreover, 
improvements on activation behavior —such as introducing 
sparse regularization for ReLU —have been shown to enhance 
generalization and convergence stability  [25]. 
3.2. The structure optimization of CNN  
In view of the difficulty of convolutional neural network, 
this paper uses gray relational analysis to determine the number 
of feature graphs. The key of this process is to use the gray 
relational analysis method to calculate the correlation degree 
between hidden nodes and output nodes, and sort these hidden 
nodes to retain those with high correlation degree. The hidden 
nodes with smaller degrees are del eted to ensure the final 
network structure can have better recognition accuracy while 
maintaining higher training performance and higher system 
stability and adaptability.

4. Experiment And Evaluation  
Based on the optimization of convolutional neu ral network, 
taking the road traffic sign as an example, the experiment of the 
optimized convolutional neural network in image recognition is 
studied.

The German Traffic Sign Recognition Benchmark (GTSRB) 
dataset [26] is used to evaluate the proposed method. There are 
totally 43 categories of traffic signs in the GTSRB. The dataset 
has been divided into training set and test set, whose sizes are 
39209 an d 12630. As is shown in Figure 4.

After performing the following preprocessing, the selected 
6,000 sample images are used as the graphic database for this 
convolutional neural net work experiment.  
1) Binarization  
Traffic sign images in GTSRB image database are shot 
randomly. In order to facilitate the processing, it is necessary to 
binarize the whole dataset for efficient information extraction and 
image recognition.  
The binarization m ethod involves  setting a threshold value, 
setting  pixels in the image that are less than the threshold to 0 , 
and setting the  others to 1.  
2) Uniform pixel size  
The image samples to be processed are stretched and 
compressed so that the 6000 samples to be processed have the 
same pixel row and column size to facilitate the experiment.

4.4. Testing For Optimized Convolutional Neural Network

The entire testing process is divided into three steps: 
network structure training, feature extraction and network t esting.

classification label.  
b. Initialization of network parameters and learning rate. 
According to the above study, the numb er of layers of 
convolutional neural network is set as 4(1 -8-18-10), and the 
number of batch samples is set to 40 and the number of iterations

number of hidden feature maps.  
c. Take batch sample s from the graphic experiment library 
and train the batch samples using the convolutional neural 
network to obtain the entire actual output vector and the output

d. Calculate the output error of the entire network and the

e. Calculate the adjustment 
  and the bias

5. Conclusion  
Image classification is an important field in pattern 
recognition. It is necessary to find an image classification method 
that can be used universally and has good performance on 


6 
different classification problems. At present, the image 
classification metho ds are mainly to extract the image 
information by manual feature extraction method. For different 
image classification problems, we need use different feature 
extraction methods, and the most effective method of a certain 
class is not necessarily valid on other issues. The emergence of 
convolutional neural network brings a new research idea to image 
classification. It adopts local experience in structure, weight 
sharing and down -sampling technology to take full account of the 
translation, rotation and zoom invariance of the image in space. 
Even if the attitude and shape of the object have changed, the 
recognition task can still be completed. In addition, it combines 
the feature extraction and classification of the traditional image 
classification technology,  and the image is classified by using the 
hierarchical structure to gradually abstract the implicit features of 
the original image.
