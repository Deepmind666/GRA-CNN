# Submission Package

`ivc_final/` is the complete extracted final submission package for Image and
Vision Computing. It includes the manuscript source bundle, final PDFs,
cover letter, highlights, graphical abstract, figures, tables, checklist, and
technical-check response.

Use `paper/` for the lightweight reviewer-facing manuscript assets. Use
`submission/ivc_final/` when the exact portal upload bundle or patent-writing
source text is needed.
