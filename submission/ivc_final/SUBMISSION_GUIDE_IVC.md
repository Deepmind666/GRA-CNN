# Image and Vision Computing Submission Guide

## Target
- Journal: Image and Vision Computing
- Publisher: Elsevier
- Primary submission entry: `https://submit.elsevier.com/IMAVIS`
- Legacy Editorial Manager page: `https://www.editorialmanager.com/IMAVIS`
- Route: subscription / non-OA

## Current Portal Note
- The `editorialmanager.com/IMAVIS` landing page currently shows an "under development" message and directs live submissions to Elsevier's new submission route.
- Use `submit.elsevier.com/IMAVIS` as the live entry point unless the journal site redirects you elsewhere on the submission day.

## Manuscript Package
- `manuscript.tex`
- `manuscript.pdf`
- `references.bib`
- `elsarticle.cls`
- `elsarticle-num.bst`
- `cover_letter.pdf`
- `highlights.txt`
- `graphical_abstract.png`
- `supplement.pdf`
- `figures/*`
- `tables/*`
- `reviewer_shortlist.md` (internal helper, do not upload unless explicitly requested)

## Internal Source Files
- Main manuscript: `manuscript_ivc.tex`
- Supplement: `supplement_ivc.tex`
- Cover letter: `cover_letter_ivc.tex`
- Highlights: `highlights.txt`
- Graphical abstract: `graphical_abstract.pdf`, `graphical_abstract.png`

## Recommended Editorial Manager Upload Types
- Manuscript: `manuscript.tex`, `references.bib`, `elsarticle.cls`, `elsarticle-num.bst`, `tables/*`
- Figure: files in `figures/`
- Highlights: `highlights.txt`
- Graphical Abstract: `graphical_abstract.png`
- Cover Letter: `cover_letter.pdf`
- Supplementary Material: `supplement.pdf`
- Do not upload `reviewer_shortlist.md`; use it only to fill the reviewer-suggestion form manually.

## Pre-Submission Checks
- Manuscript compiles cleanly under `elsarticle`
- Supplement compiles cleanly
- Abstract length is within the IVC limit
- Keywords count is valid
- No previous-journal or previous-publisher wording remains
- No Tiny pilot wording remains
- No second-best underlining remains in IVC tables
- Graphical abstract is present and non-AI-derived
- The manuscript includes an AI-use disclosure before the references
- A declarations form generated from Elsevier's official declarations tool still needs to be uploaded through the portal
