# Paper Assets

This directory contains the paper assets associated with the submitted IVC version of GRA-CNN.

- `manuscript.pdf`: main manuscript
- `supplement.pdf`: supplementary material
- `figures/`: publication figures used in the submission package
- `tables/`: publication-ready LaTeX tables used by the manuscript and supplement
- `tables/source_csv/`: source CSV tables for local evidence checks
- `source_tools/`: selected scripts and editable sources used to regenerate final figures/tables

These files are included to support manuscript review and result traceability.

The complete extracted submission package is stored in `submission/ivc_final/`.
