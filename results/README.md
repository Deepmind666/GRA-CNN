# Filtered Results

This directory contains filtered run-level outputs used to support the submitted GRA-CNN manuscript.

Included:

- seed-level JSON records
- derived CSV summaries
- selected LaTeX result tables
- selected short analysis notes directly tied to the paper

Excluded:

- launcher logs
- local checkpoints
- lock files
- private review notes
- workstation-specific diagnostics

The goal is to provide enough evidence for paper-level reproducibility without publishing the full internal working directory.
